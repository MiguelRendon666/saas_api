# Auth Service

Microservicio de autenticación y gestión de usuarios.

## Modelos incluidos:
- Usuario
- Rol
- UsuarioRol
- Permiso
- PermisoAsignado

## Base de datos
- Base de datos: `auth_service_db`
- Tablas compartidas: `Base`

## Instalación
```bash
pip install -r requirements.txt
```

## Configuración
Copiar `.env.example` a `.env` y configurar las variables de entorno.
