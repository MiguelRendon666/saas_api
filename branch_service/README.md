# Branch Service

Microservicio de gestión de turnos y cortes de caja.

## Modelos incluidos:
- TurnoSucursal
- CorteCaja

## Base de datos
- Base de datos: `branch_service_db`
- Tablas compartidas: `Base`

## Instalación
```bash
pip install -r requirements.txt
```

## Configuración
Copiar `.env.example` a `.env` y configurar las variables de entorno.
