# Inventory Service

Microservicio de gestión de inventarios.

## Modelos incluidos:
- StockSucursal
- ListaPrecios
- InventarioLotes

## Base de datos
- Base de datos: `inventory_service_db`
- Tablas compartidas: `Base`

## Instalación
```bash
pip install -r requirements.txt
```

## Configuración
Copiar `.env.example` a `.env` y configurar las variables de entorno.
