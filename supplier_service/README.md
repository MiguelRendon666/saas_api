# Supplier Service

Microservicio de gestión de proveedores.

## Modelos incluidos:
- ProveedorEmpleado
- ProveedorEmpresa
- ProveedorMarca
- ProveedorProducto

## Base de datos
- Base de datos: `supplier_service_db`
- Tablas compartidas: `Base`, `BaseContactoObject`

## Instalación
```bash
pip install -r requirements.txt
```

## Configuración
Copiar `.env.example` a `.env` y configurar las variables de entorno.
